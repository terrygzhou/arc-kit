What is ArcKit?

ArcKit is an AI harness that wraps your AI coding assistant: it gates, traces, and provenance-stamps everything the assistant produces. From stakeholder analysis and risk registers to design reviews and traceability matrices, every artifact is template-driven and audit-ready. A UK Government baseline ships alongside community-contributed jurisdiction and sector overlays (EU, France, Austria, Canada, UAE, Australia, USA, UK Finance, UK NHS, UK G-Cloud).

Systematic

Follow GDS Agile Delivery phases with a structured command workflow. Every artifact is version-controlled, traceable, and linked to business drivers.

Multi-Jurisdiction

UK Government core (TCoP, Secure by Design, GDS Service Standard, MOD JSP 936, HM Treasury Green/Orange Books) with community overlays for GDPR, AI Act, NIS2, ANSSI, PDPL, and more.

AI-Assisted

Works with Claude Code, Gemini CLI, Codex CLI, OpenCode CLI, and GitHub Copilot. AI generates first drafts from templates; architects review and refine.

How it fits together

Your AI coding assistant sits at the bottom. Every tool call is intercepted by The Enterprise Governance Harness — gates block unsafe writes, the core engine drives generation, sector overlays and live MCP knowledge compose in, and observers stamp provenance and traceability. What comes out the top are governed, audit-ready artifacts.

UK Government

Government Code Discovery

Search 24,500+ UK government repositories before building from scratch. Powered by govreposcrape, no API key required.

gov-code-search

Natural language search across government repositories. Find how other departments solved similar problems.

gov-reuse

Systematic reusability assessment. Score candidates on licence, quality, documentation, tech stack, and maintenance.

gov-landscape

Map the government code landscape for a domain. Identify who built what, common patterns, and collaboration opportunities.

Supports TCoP Point 3 ("Be open and use open source") and the CDDO "reuse before build" principle. Existing research commands now include a government reuse check as a fifth build-vs-buy option.

Interoperability

Open Knowledge Format exchange

Share ArcKit architecture knowledge as portable Markdown bundles with OKF-compatible frontmatter, or import external OKF bundles as reviewable research notes.

export-okf

Copy ArcKit ARC-*.md artifacts into an OKF bundle with portable metadata. Source artifacts stay unchanged.

import-okf

Classify OKF Markdown, write an import report, and materialize safe entries as RSCH review notes.

Opt-in stamping

Native ARC frontmatter stamping is disabled by default and enabled only through explicit project config or environment.

Works with your AI

Every core command and community overlay produces identical outputs across AI systems. Choose the platform that fits your workflow.

Claude Code

Premier

Full experience with agents, hooks, MCP servers, and output validation.

/plugin marketplace add tractorjuice/arc-kit

Gemini CLI

Full

Native extension with all commands, templates, scripts, and MCP servers.

gemini extensions install tractorjuice/arckit-gemini

OpenAI Codex CLI

Core

Commands and templates copied into your project via CLI.

arckit init my-project --ai codex

OpenCode CLI

Core

Commands and templates with flexible model selection.

arckit init my-project --ai opencode

GitHub Copilot

Core

Prompt files and agents for VS Code Copilot Chat.

arckit init my-project --ai copilot

How ArcKit tidies Wardley Map labels: a deterministic placement engine, stepped through

Mermaid draws every wardley-beta component label at the same offset, so clustered components on a real map land on top of each other. ArcKit fixes this with a PostToolUse hook: a pure, deterministic engine that projects the map to pixels, generates 32 candidate slots per label, scores collisions, places the most-constrained labels first, and converges to an idempotent result.
